MOHAA 100m RACE MOD - INSTALLATIE
=====================================

STAP 1: Bestanden kopiÃ«ren
---------------------------
Kopieer de BESTANDEN uit de 'main' folder naar je MOHAA 'main' folder.

BRONLOCATIE:
  mohaa_race_mod\main\dm_race.scr
  mohaa_race_mod\main\race_100m.scr
  mohaa_race_mod\main\race_config.scr
  etc...

DOELLOCATIE (standaard):
  C:\Program Files (x86)\EA Games\Medal of Honor Allied Assault\main\dm_race.scr
  C:\Program Files (x86)\EA Games\Medal of Honor Allied Assault\main\race_100m.scr
  C:\Program Files (x86)\EA Games\Medal of Honor Allied Assault\main\race_config.scr
  etc...

LET OP: Kopieer de BESTANDEN, niet de hele 'main' folder!
Je wilt NIET: C:\...\MOHAA\main\main\dm_race.scr (fout!)
Je wilt WEL: C:\...\MOHAA\main\dm_race.scr (goed!)

STAP 2: Server starten
----------------------
1. Start MOHAA
2. Druk op ~ om de console te openen
3. Type: sv_maxclients 2
4. Laad een bestaande DM map, bijvoorbeeld:
   - map dm/mohdm1  (Stalingrad)
   - map dm/mohdm2  (Northern Italy)
   - map dm/mohdm3  (Destroyed Village)
5. Type: exec global/dm_race.scr::main

Alternatief via dedicated server:
----------------------------------
Maak een .bat bestand met:
mohaa_server.exe +set dedicated 1 +set sv_maxclients 2 +map dm/mohdm1 +exec global/dm_race.scr::main

STAP 3: Spelen
--------------
1. Twee spelers joinen de server
2. De race start automatisch wanneer beide spelers aanwezig zijn
3. Beide spelers worden naar de startlijn geteleporteerd
4. Countdown: 3-2-1-GO!
5. Ren 100 meter naar de finish lijn
6. Eerste speler die de finish bereikt wint de ronde
7. Best of 5 rondes - eerste bij 3 punten wint de match

CONTROLS:
---------
- W/A/S/D of Pijltjestoetsen = Bewegen
- Spatie = Springen

TIPS:
-----
- Blijf rechtdoor rennen voor de kortste route
- Spring niet teveel, het vertraagt je
- Sprint is niet beschikbaar in vanilla MOHAA scripting

CONFIGURATIE:
-------------
Je kunt de race aanpassen in main/race_config.scr:
- Race afstand (standaard 100m = 5000 units)
- Sprint snelheid en stamina
- Aantal rondes om te winnen

TROUBLESHOOTING:
----------------
- Als de scripts niet laden, controleer of de bestanden in de juiste 'main' folder staan
- Zorg dat sv_maxclients op minimaal 2 staat voor multiplayer
- Bij crashes, verhoog je heap size: +set com_hunkmegs 128
- Werkt op elke DM map - geen custom map nodig!
